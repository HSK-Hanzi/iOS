# App Feature Ideas

## Learning

- [x] Configure your HSK level; drill words and characters
- [x] Fully configure your study mode: Drill English->Chinese or Chinese->English, drill pinyin or Hanzi (or both), drill pronunciation or strokes
- [x] Flashcard view with excellent physicalist reversal animation
- [x] Information-rich, well-typeset dictionary view with tappable links
- [x] Tapping a Hanzi phrase brings up the pinyin + definition as tooltip with option to travel to that dictionary entry

## Stroke Drawing View

- [ ] Custom stroke drawing using simulated pen/brush tips (use opaque types?) -- calligraphy brush, ballpoint pen, etc. (will maybe need to reverse engineer brush pressure using stroke width?)
- [ ] Natural stroke animation using humanistic easing
- [x] Test your stroke drawing -- marks strokes red or green if you do them in the correct order and correct direction (with animated hints if you screw up)
- [ ] Apple Pencil support, leveraging brush/pen tips with weight

## Settings Page

- [ ] Stroke brush head, simplified/traditional, HSK level

## Practice

- [x] Import sentences/convos from HSK workbooks and include them in the Practice view

## Quiz

- [x] Writing quiz mode: It shows you a character and then you have to draw it. Same config options as the flashcard quiz mode.
- [x] Listening quiz mode: One of the HSK sentences is played and you have to transcribe the pinyin properly.
